# Foobar
La aplicacion de desarrollo con Laravely Vuejs son los frameworks que mejor manejo decidi usarlos para demostrar la mayor
capacidad de mi conocimiento en el desarrollo.
Viendo que el desarrollo era de alcance abierto, decidi por cuenta propia crear una aplicación de transporte en la cual
podra administrar ciudades propietarios conductores vehiculos, y asignar rutas de viajes, adicional podra sacar reportes
de algunos modulos, espero que os guste
## Installation

- descargar desde el repositorio con el comando: git clone https://github.com/castagnola/app-prueba.git
- crear una base de datos 
- copiar el archivo .env.example llamarlo .env
- Configurar la conección a la Base de Datos en el .env
##Comandos
```bash
composer install
```
```bash
php artisan key:generate
```
```bash
php artisan migrate
```
```bash
php artisan passport:install
```
```bash
php artisan db:seed 
```
```bash
php artisan serve
```
